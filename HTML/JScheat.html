Beginner JavaScript Cheatsheet

Element Description Example
for A loop statment best used when you know how many times you wanna loop for(let i = 0; i < x; i++){}
while A loop statment that checks a condition and will loop until that condition is met/ no longer true while(x === 2){}
do while Like the while loop but checks the condition after looping and not before do{}while(x === 2);
console used to target the console console.log(x);
document used to target the document document.getElementById();
function used to declare a function function sumNumber(){}
alert creates a text popup alert("hello world");
Element Description Example
prompt creates a prompt to enter text/ numbers prompt("enter your name";
== checks if the 2 values are equal (doesnt matter if string or integer) 5 == '5'(true)
=== checks if values are equal in value and type 5 === '5' (false)
&& the AND operator 5 == 5 && '5' === '5'
|| The OR operator 5 == '5' || 5 === '5'
if A conditional statement that will execute only if the condition is true if(x === 8){}
switch Like the if statement but you can easily check multiple conditions using a case, must use break to prevent fall through switch(x){
case 1:
break;}
let declares a variable(string or int or bool) let myName = 'Braydon'
const declares a constant variable const myName = 'Braydon'
