CSS Cheatsheet For Newbies And Pros!

Element Description Example
display sets the display type of an element(flex, box, etc) div{display:flex}
#idname targets the element with the id "idname" #idname{style changes to that specific element}
.classname targets all elements with the class "classname" .classname{style changes to all elements with that class}
flex-direction sets the direction of flexible items div{display:flex
flex-direction: column}
font-size Sets the size of a font, either with a variable or with keywords like "small", "larger", etc. p{font-size: larger}
font-family sets the font style with an option to add multiple incase the inital font style doesnt work p{font-family: arial "Times New Roman"}
align-content Used with the flex property, used to align content within a flex container div{display: flex;
align-content: center;}
align-items align items within a container div{align-items: center;}
Element Description Example
align-self aligns selected item p{align-self: auto;}
div:last-child selects the last child of its parent(in this case it would be the last div of its parent) div:last-child{style changes}
div:first-child selects the first child of its parent(in this case it would be the first div of its parent) div:first-child{style changes}
div:nth-of-type(3) selects an element based on its position among siblings of the same type(in this case it would be the 3rd div) div:nth-of-type(3){style changes}
:root similar to the html selector but is higher priority, also usually used to declare custom variables :root{--primary-color: blue;}
a:hover a link when you hover over it a:hover{color: red;}
a:visited link after it has been visited a:visited{ background-color: green;}
a:active a link that is being activated a:active{ color: aqua;}
